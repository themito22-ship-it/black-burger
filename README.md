# Black Burger — Site

Site institucional/cardápio da Black Burger, feito em HTML, CSS e JavaScript puro (sem build, sem framework).

## Como publicar no Vercel

Veja o passo a passo completo na conversa com o Claude, ou resumidamente:

1. Suba estes arquivos para um repositório no GitHub.
2. Entre em https://vercel.com e faça login com sua conta do GitHub.
3. Clique em "Add New Project" e selecione este repositório.
4. Não precisa mudar nenhuma configuração — clique em "Deploy".
5. Pronto! O Vercel te dá um link tipo `black-burger.vercel.app`.

## Antes de publicar, troque:

- O número de WhatsApp `5511999999999` (aparece 4 vezes no `index.html`)
- Endereço e horário de funcionamento (seção "Localização")
